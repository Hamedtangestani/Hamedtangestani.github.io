About Me

I'm a Graduate Student Researcher at UC Santa Cruz, specializing in robotics and embedded systems. My research focuses on developing intelligent robotic systems using advanced machine learning techniques including imitation learning and reinforcement learning.

I have hands-on expertise in LeRobot, MuJoCo, ROS2, PyTorch, and hardware prototyping. My work spans from building complete robotic platforms to developing sophisticated control algorithms and simulation-to-real pipelines.

6-DoF Robotic Arm with LeRobot

LeRobot PyTorch RL Imitation Learning

Intelligent Joint Simulator

Built an intelligent joint simulator using graph theory and computer vision, automatically generating physics-accurate MuJoCo models from real-time hand/body tracking for robotic control.

MuJoCo Computer Vision Graph Theory Simulation

Desktop CNC Machine

Designed and implemented a desktop CNC machine using the coreXY motion system and ATmega328 microcontroller, developing custom firmware for 2D path control and fabricating mechanical parts.

Embedded Systems ATmega328 CoreXY Firmware

ROV Sensor Integration RS485 Navigation
